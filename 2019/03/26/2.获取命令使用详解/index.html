2.获取命令使用详解

发表于 2019-03-26 | 更新于 2019-04-08 | 分类于 linux


命令的语法通用格式、获取命令帮助文档

命令的语法通用格式

COMMAND（命令） OPTIONS（选项） ARGUMENTS（参数）
发起一个命令：请求内核将某个二进制程序运行为一个进程
命令本身是一个可执行文件：二进制文件，有可能会调用共享库文件；大多数文件会存放在/bin、/sbin、/usr/bin、/usr/sbin;非标准系统自带的/usr/local/bin、/usr/local/sbin
共享库文件：/lib、/lib64、/usr/lib、/usr/lib64、/usr/local/lib、/usr/local/lib64;
命令文件格式：exe、msi、ELF（Linux）
命令分类：由shell程序自带的命令（内置命令builtin）；独立的可执行程序文件，文件名及命令名（外部命令）；type COMMAND查看命令类型

获取命令帮助文档

  • 内部命令
    help COMMAND只能显示内部命令的帮助信息
  • 外部命令
    （1）命令自带简要格式的使用帮助：COMMAND –help
    （2）使用手册：man；存在与/usr/share/man下，该路径下存在不同的章节；
       man COMMAND
       NAME：功能行说明
       SYNOPSIS：语法格式；[ ]可选内容；< >必填内容；a|b|c多选一；…..同类型可出现多个
       DESCRIPTION：描述
       OPTIONS：选项
       EXAMPLES：使用实例
       AUTHOR：作者
       BUGS：报告程序bug方式
      /usr/share/man下的章节之分：
       1：用户命令
       2:系统调用
       3:C库调用
       4:设备文件及特殊文件
       5:文件格式（配置文件格式）
       6:游戏使用帮助
       7：杂项
       8：管理工具及守护进程
      man CHAPTER COMMAND 查看指定章节的帮助手册
      whatis COMMAND 显示命令在那些章节；其执行过程是查询数据库进行的；使用makewhatis来手动更新数据库
      man命令手册中的操作方法：
    • 翻屏
         空格：向文件尾部翻一屏
         b：向文件首部翻一屏
         回车：向文件尾部翻一行
         k：向文件首部翻一行
         G：跳转到最后一行；#G跳转到第#行
    • 文本搜索
         /KEYWORD：从文件首部向文件尾部进行搜索。不区分大小写
         ？KEYWORD：从文件尾部向文件首部进行搜索
         以上两种搜索关键词定位：n 与搜索顺序相同；N 与搜索顺序相反
    • 退出 q
