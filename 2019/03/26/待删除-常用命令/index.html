发表于 2019-03-26 | 更新于 2019-04-10 | 分类于 linux

常用命令

常用命令

  • pwd
    显示当前工作目录
  • cd
    切换工作目录，内嵌命令，不带参数时切换到用户的家目录
    cd ~USERNAME：切换至指定用户的家目录
    cd -：切换至上一次的目录
  • ls
    列出指定目录下的内容
    ls [OPTIONS]…[FILE]…
    -a :显示所有文件包括隐藏文件
    -A :显示所有文件，除..与.
    -l:显示长格式列表
    -h：文件大小单位换算
    -d：查看目录本身，而非其内部的文件列表
    -r:逆序显示
    -t：按照文件更改时间进行排序
    -R:递归显示
  • cat
    文本查看工具
    cat [OPTIONS]…[FILE]…
    -n:给显示的文本行编号
    -E：显示行结束符$
  • tac
    与cat显示顺序相反
    tac [OPTIONS]…[FILE]…
    -n:给显示的文本行编号
    -E：显示行结束符$
  • file
    查看文件内容类型
    file [FILE]…
  • echo
    回显
    回显的STRING使用单引号时为强引用，直接展示STRING；双引号为弱引用，可替换变量为变量的值；反引号，反引号中的内容为命令，回显内容为命令的结果
    echo [OPTION]…[STRING]…
    -n：不进行换行
    -e：转义符生效

  • shutdown
    关机
    shutdown [OPTION]… TIME [MESSAGE]

    • OPTION
      -h:关机
      -r:重启
      -c：取消关机或重启
    • TIME
      now
      hh：mm
      +10 ：10min之后；+0相当于now
    • MESSAGE
      要传送给所有登入用户的信息

  • date
    显示或设置系统时间与日期
    -s:设置系统时间

  • hwclock
    显示硬件时钟，设置硬件时钟的时间和设置硬件时钟为系统时间，也可设置系统时间为硬件时钟的时间
    -w:将硬件时间修改为系统时间，时间以系统时间为准
    -s：将系统时间修改为硬件时间，时间以硬件时间为准

  • which
    查找指定命令的可执行文件的绝对路径；查找的机制是在 $PATH中进行查找
    which [OPTION] COMMAND
    -v：现实命令的版本
    -- skip-alias：忽略别名

  • whereis
    查找指定命令的可执行文件的绝对路径；查找机制通过文件索引数据库来查找的

  • who
    查看当前已登录的用户信息

    • whoami、who am i、who三者区别
      whoami；显示的是当前用户下的用户名
      who am i：显示的是登录时的用户名
      who：显示当前真正登录系统中的用户（不会显示那些用su命令切换用户的登录者）

  • w
    目前登入系统的用户有那些人，以及他们正在执行的程序

  • type
    查看命令类型

  • alias
    查看所有可用别名定义 【别名与原名相同，此时原名被隐藏，使用 \COMMAND 可执行原命令】
    alias NAME=‘COMMAND’：定义别名，只对当前用户有效
    unalias NAME：撤销别名
