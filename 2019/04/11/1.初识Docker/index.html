Docker

Docker基本概念

Docker镜像（Image）

  Docker镜像就是一个只读的模板，用于创建Docker容器。Docker有简单的机制来创建镜像或更新现有镜像，也可以将其他人的镜像拿来直接使用。

Docker容器（Container）

  Docker利用容器来运行应用。容器是从镜像创建的运行实例。可以被启动、停止、删除等。每个容器是相互隔离的。

Docker仓库（Repository）

  仓库是集中存放镜像文件的场所。仓库分为公开仓库（Public）和私有仓库（Private）两种形式。最大的仓库是Docker Hub；国内也有去多的公开仓库，可以更快速访问。
  用户也可以在本地创建私有仓库。当用户创建了自己的镜像后可以使用push命令将上传到私有或公开仓库，这样另外一台机器需使用这个镜像的时候就可以直接pull下来。

安装Docker

  Docker要求CentOS系统的内核版本高于3.10且是64位系统，可以使用uname -r进行查看。
