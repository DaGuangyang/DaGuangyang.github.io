6.linux文件与目录

发表于 2019-04-08 | 更新于 2019-04-09 | 分类于 linux


文件、目录

文件类型

-：常规文件，即f
d：目录文件
b：块设备，以“block”为单位进行随机访问
c：字符设备，以“character”为单位进行线性访问
 对同一类设备进行区分;主设备号：标识设备类型，进行而确定要加载的驱动程序;次设备号：用于区分同一类型的不同设备
l：符号链接文件（软链接文件），类似Windows中的快捷方式
p：命名管道
s：套接字文件

目录的相关操作

cd
切换工作目录
 . ：表示上层目录
 .. :表示当前目录
 -：表示前一个工作目录
 ~：表示当前用户的家目录
 ~ yyg：表示yyg用户的家目录

pwd
显示当前工作目录路径
 -p：显示软连接指向文件的路径，而非软连接文件的路径

mkdir
创建新的目录
 -m：在创建目录时设置目录权限
 -p：递归创建目录

rmdir
删除空目录
 -p：递归删除空目录

查看文件与目录的信息

ls
 -a：显示所有文件，包括隐藏文件
 -l：显示文件的长信息
 -t：以时间进行排序
 -r：反向排序
 -d：仅列出目录本身，而非目录下的所有文件

tree
以树的形式递归展示当前目录下的文件和目录，系统中可能没有该命令，使用yum install tree来进行安装

stat FILE
显示文件状态
access：最近访问时间
modify：最近修改时间
change：最近状态改动时间

文件与目录的复、删除、移动

cp
 -p：连同档案的属性（权限、时间、用户）一并复制
 -r：递归复制
 -i：当目标目录下有同名文件时进行提示

rm
删除
 -r：递归删除
 -f：强制删除，忽略不存在的文件
 -i：提示确认

mv
移动或重命名
 -f:强制移动
 -i：提示确认

取得路径的文件名或目录名

basename
 取得文件或目录的基本名称
dirname
 取得文件或目录的目录名

文件内容查看

直接展示文件内容

cat
从文件第一行开始显示
 -n：列出行号，空白行也有行号

tac
从文件的最后一行开始倒序显示

翻页展示文件内容

more
按键：
 space：向下翻一页
 Enter：向下翻一行
 b：往回翻页
 /STRING：从当前向下查找字符串
 q：退出

less
与more相比可以向上翻页
按键：
 space：向下翻一页
 pagedown：向下翻一页
 pageup：向上翻一页
 /STRING:从当前向下查找字符串
 ?STRING:从当前向上查找字符串
 n：重复上一次的搜索（方向与/或？相同）
 N：反向重复上一次的搜索（方向与/或？相反）
 g：跳转到文件第一行
 G：跳转到文件的最后一行
 q：退出

按行截取文件内容展示

head
取出前面几行
-n NUM：后面接数字，表示显示NUM行

tail
取出后面几行
-n NUM：后面接数字，表示显示NUM行
-f：持续检测文件，在查看日志文件时常用

查看非纯文本文件

od

修改文件时间或创建空文件

touch [选项]
选项：
 -a：修改atime为当前时间，access time时间，访问时间
 -m：修改mtime为当前时间，modify time时间，内容修改时间
 -c：仅修改文件时间，若文件不存在则不创建文件
 -t：指定修改的时间，格式[YYYYMMDDhhmm]

查看文件类型

file FILE

搜索文档

which COMMAND
显示外部命令文件完整路径名，该命令是根据\$PATH路径进行查找，所以只能查看外部命令的命令文件
还能查看到该命令的别名

whereis [选项] COMMAND
显示命令完整路径，手册页路径
选项：
 -b:只显示命令的二进制程序路径
 -m：只显示命令的手册页路径
