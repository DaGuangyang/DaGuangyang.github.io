linux中的权限管理

权限管理


进程访问文件访问权限说明：进程的属主与文件的属主是否相同，若相同则应用属主权限；否则，则检查进程的属主是否属于文件的属组，若相同则应用属组权限；否则使用other权限
—|r：读|w：写|x：执行
:—:|:–:|:—:|:—:
文件|获取文件数据|修改文件数据|可将此文件运行为进行
目录|可使用ls查看目录下的文件列表|可创建、删除此目录下的文件列表|可cd至此目录中，可使用ls -l查看文件详情

权限管理命令

三类用户：u—属主 g—属组 o—其他 a—所有

chmod
修改权限，用户只能修改属主为自己的文件权限
-R：递归修改权限
（一）chmod u=rwx FILE
   chmod 777 FILE
   chmod u+x FILE
（二）chmod --referece=LIKE_FILE FILE 将FILE的权限修改为LIKE_FILE权限一样

chown
修改属主、属组，仅管理员可用
-R：递归修改
（一）chown 属组[:属组] FILE
（二）chown --referece=LIKE_FILE FILE 将FILE的属主属组修改为LIKE_FILE权限一样

chgrp
修改属组，仅管理员可用
-R：递归修改
（一）chgrp 属组 FILE
（二）chgrp --referece=LIKE_FILE FILE 将FILE的属组修改为LIKE_FILE权限一样
