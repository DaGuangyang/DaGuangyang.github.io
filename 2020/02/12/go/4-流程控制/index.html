Go 流程控制

if

if判断基本写法

注意：左右大括号的位置要严格按照该位置来进行，否则将会出现编译错误

if判断特殊写法

可以在if语句中定义局域变量

这样定义的变量只能在该if判断中使用，是一个局域变量。

for循环

Go 语言中的所有循环类型均可以使用for关键字来完成。
初始语句–>条件表达式为True–>执行循环体–>执行结束语句–>……–>条件表达式为False时结束循环。

for循环基本格式

当条件表达式为True时进行下一次循环，当条件表达式为False时结束循环。

for 循环的初始语句可以不写，但是初始语句后的 分号必须要保留。

for循环的初始语句和结束语句可以都不写。此时可以将分号都省略。

for循环的无限循环

可以使用break（跳出循环）、continue（跳过本次循环，执行下一次的循环）

for range循环（键值循环）

Go语言中可以使用for range遍历数组、切片、字符串、map 及通道（channel）。 通过for range遍历的返回值有以下规律：

  1. 数组、切片、字符串返回索引和值。
  2. map返回键和值。
  3. 通道（channel）只返回通道内的值。

switch case

  • 使用switch语句可方便地对大量的值进行条件判断。

switch 中只能有一个default一个分支。

  • 分支还可以使用条件表达式来进行判断,switch后不需要跟判断变量。

  • fallthrough语法可以执行满足条件的case的下一个case（了解就可以）。

输出

goto 跳转到指定标签

goto语句在跳出多层循环时可以简化代码。

break 跳出循环

break可以结束for,switch,select代码块。
break [标签] 这样还可以跳出指定标签的代码块，标签要求必须在对应的for,switch,select的代码块上。

continue 继续下次循环

continue可以结束当前循环，直接执行下次循环，但只能在for循环中使用。
continue后加标签，代表开始标签对应的循环。
