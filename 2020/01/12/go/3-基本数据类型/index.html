Go语言的3-基本数据类型

基本数据类型

整型

类型 描述
uint8 无符号 8位整型 (0 到 255)，别名byte
uint16 无符号 16位整型 (0 到 65535)
uint32 无符号 32位整型 (0 到 4294967295)
uint64 无符号 64位整型 (0 到 18446744073709551615)
int8 有符号 8位整型 (-128 到 127)
int16 有符号 16位整型 (-32768 到 32767)
int32 有符号 32位整型 (-2147483648 到 2147483647)，别名rune
int64 有符号 64位整型 (-9223372036854775808 到 9223372036854775807)
类型 描述
uint 32位操作系统上就是uint32，64位操作系统上就是uint64
int 32位操作系统上就是int32，64位操作系统上就是int64
uintptr 无符号整型，用于存放一个指针

注意：

  • 在使用int、uint类型时，不能假定它是32位或64位整型，而是考虑int、uint可能在不同平台上的差异。
  • 获取对象的长度的len()函数返回的长度可以根据不同平台的字节长度进行变化。实际使用中，切片或map的元素数量等都可以使用int来表示。在涉及到二进制传输、读写文件的结构描述时，为保持文件的结构不会受到不同编译目标平台字节长度的影响，不要使用int、uint。

二进制&八进制&十六进制

浮点型

Go语言中支持float32、float64两种浮点型，Go语言中默认的小数都是float都是float64类型。

打印浮点型时，可以使用%f。

布尔值

Go语言中以bool类型进行声明布尔型数据，布尔型数据只有true和false两个值。
注意

  1. 布尔型变量默认为值为false
  2. Go语言中不允许将整型强制转换为布尔型
  3. 布尔型无法参与数值运算，也无法与其他类型进行转换

fmt总结

%c对应字符

字符串

Go语言中的字符串内部使用UTF-8编码，字符串的值必须使用双引号""进行标记，可以在Go语言代码中直接进行添加；单引号包裹的是字符（单个字母、单个汉字）。

字符串转义符

转义符 含义
\r 回车符（返回行首）
\n 换行符（直接跳到下一行的同列位置）
\t 制表符
\’ 单引号
\” 双引号
\\ 反斜杠

多行字符串

Go语言中定义多行字符串时，必须使用反引号包裹。

反引号中的换行将被作为字符串中的换行，但是所有的转移字符将无效，文本将原样输出。

字符串的常用操作

方法 介绍
len(str) 求字符串长度
+或fmt.Sprintf 拼接字符串,fmt.Sprintf不输出，而是需要一个变量来接收拼接后的值
strings.Split(STRING,CHAR) 以CHAR来分割STRING分割，需要一个变量来接收
strings.Contains(STRING1,STRING2) 判断STRING1是否包含STRING2，返回布尔值
strings.HasPrefix(STRING1,STRING2),strings.HasSuffix(STRING1,STRING2) 判断STRING1前缀/后缀是否为STRING2
strings.Index(STRING1,STRING2),strings.LastIndex(STRING1,STRING2) STRING1中STRING2出现的位置
strings.Join(a[]string, sep) join操作,将列表进行拼接，间隔符为sep

byte和rune类型

Go语言中字符有两种：

  • uint8（byte）：代表了ASCII码的一个字符。
  • rune(uint8)：代表一个UTF-8字符。
    所以当需要处理中文时，则需要用到rune类型，rune实际是一个int32。

修改字符串

要修改字符串，首先要先将字符串转换为 []rune 或 []byte，修改后再转换为字符串类型。

类型转换

Go语言中只有强制类型转换。
