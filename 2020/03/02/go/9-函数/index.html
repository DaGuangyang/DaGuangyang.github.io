Go 函数

函数

函数定义

Go语言中函数使用func关键字：

函数名:由数字、字母、下划线组成。但是函数名的第一个字母不能是数字。在同一个包内，函数名也不能重名。
参数：参数由参数名、参数类型构成，多个参数之间使用逗号隔开。
返回值：返回值由返回变量和其变量类型组成，也可以只写返回值的类型。
参数和返回值都是可选的，一个函数可以没有参数和返回值。

参数

类型简写

函数的参数中如果相邻变量的类型相同，则可以省略前一个参数的类型。

可变参数

可变参数是指函数的参数数量不固定，Go语言中的可变参数通过在参数名后加...来标识。
注意：可变参数要作为函数的最后一个参数。

返回值

Go语言中通过return关键字向外输出返回值。

多返回值

Go语言中函数支持多返回值，函数如果有多个返回值时必须用()将所有返回值包裹起来，并使用,隔开。

函数进阶

变量作用域

全局变量

全局变量是定义在函数外部的变量，它在程序整个运行周期内都有效。 在函数中可以访问到全局变量。

局部变量

局部变量分为两种：

  1. 函数内定义的变量无法在函数外使用(在for、if、switch中定义的变量同样属于局部变量)

  2. 局部变量和全局变量重名，优先访问局部变量

匿名函数和闭包

匿名函数

Go语言中函数内部不能定义函数，智能定义匿名函数。匿名函数就是没有函数名的函数。

匿名函数因为没有函数名，所以没有办法想普通函数那样调用，所以匿名函数需要保存到某个变量或作为立即执行的函数。

defer

Go语言中的defer语句会将其后面跟随的语句进行延迟处理，在defer所属的函数即将返回时，将延迟处理的语句按照defer顺序的倒序进行执行，也就是，先被defer的语句最后被执行，最后被defer的语句先执行。 一般在处理释放资源的时候进行使用，例如：资源清理、文件关闭等。

内置函数

内置函数就是不需要进行导入就可以使用的函数
| 内置函数 | 介绍 |
| :————- | :—————————————————————– |
| close | 主要用来关闭channel |
| len | 用来求长度，比如string、array、slice、map、channel |
| cap | cap 是容量的意思，用于返回某个类型的最大容量（只能用于切片和 map） |
| new | 用来分配内存，主要用来分配值类型，比如int、struct。返回的是指针 |
| make | 用来分配内存，主要用来分配引用类型，比如chan、map、slice |
| append | 用来追加元素到切片中 |
| copy | 用来复制切片 |
| panic和recover | 用来做错误处理 |

panic/recover

Go语言中没有异常机制，但是可以使用painc/recover模式来处理错误。panic可以在任何地方引发，recover只能在defer调用的函数中有效。
recover要搭配defer使用。
defer一定要在可能引发panic的语句之前进行定义。
