redis数据类型

redis支持五种数据类型：string（字符串）、hash（哈希）、list（列表）、set（集合）、zset（有序集合）。

string（字符串）

string是redis最基本的类型，一个key对应一个value，最大存储521MB。

redis string命令

命令 描述
set KEY VALUE 设置指定key值，若key已经存在，则新值会覆盖旧值
get KEY 获取指定key的值；key不存在，返回nil；值不是string类型，返回报错
getrange KEY START END 获取key值按照切片的方式截取的值(包括 start 和 end 在内)
getset KEY NEWVALUE 将key的值设置为newvalue，并返回旧的值；若无旧值返回nil；key存在但value不是string，返回报错
mget KEY1 KEY2 获取多个key的值
setex KEY TIMEOUT VALUE 对指定的key设置过期时间timeout，单位秒
setnx KEY VALUE 只有在key不存在时，才设置key的值，也就是防止覆盖原value
strlen key 返回key的值的长度
mset KEY1 VALUE1 KEY2 VALUE2 同时设置多个key-value对
msetnx KEY1 VALUE1 KEY2 VALUE2 同时设定之前不存在的key的值
incr KEY 若value为数字，则数字的值加一；否则报错；若key不存在，先将 key 的值设为 0 ，再执行加法操作
incrby KEY NUM 若value为数字，则数字的值加NUM；否则报错;若key不存在，先将 key 的值设为 0 ，再执行加法操作
incrbyfloat KEY NUM key中所储存的值加上指定的浮点数
decr KEY 将值减一
decrby KEY NUM 将值减去指定NUM
append KEY NEWVALUE 将key的值尾部加上NEWVALUE；若key不存在则值直接设定为NEWVALUE

hash（哈希）

Redis hash 是一个 string 类型的 field 和 value 的映射表，hash 特别适合用于存储对象。
Redis 中每个 hash 可以存储 232 - 1 键值对（40多亿）。

命令 描述
hdel KEY FIELD1 FIELD2 删除一个或多个哈希表字段
hexists KEY FIELD 查看哈希表中指定的field是否存在
hget 获取存储在哈希表中指定字段的值
hgetall KEY 获取哈希表中所有的字段和值
hincrby KEY FIELD NUM 为哈希表指定的字段的整数加上NUM
hincrbyfloat KEY FIELD NUM 为哈希表指定的字段的浮点数数加上NUM
hkeys KEY 获取哈希表中所有的字段
hlen KEY 获取哈希表中字段的数量
hmget KEY1 FIELD1 FIELD2 获取key指定的字段
hmset KEY FIELD1 VALUE1 FIELD2 VALUE2 同时将多个field-value对设置到哈希表中
hset KEY FIELD VALUE 将哈希表 key 中的字段 field 的值设为 value
hsetnx KEY FIELD VALUE 只有在字段 field 不存在时，设置哈希表字段的值
hvals KEY 获取哈希表中所有值

list（列表）

列表是简单的字符串列表，按照插入的顺序排序，可以在列表的头部或者尾部添加元素。

命令 描述

lindex KEY INDEX_NUM |通过索引获取列表中的元素
linsert KEY BEFORE/AFTER PIVOT VALUE | 将值 value 插入到列表 key 当中，位于值 pivot 之前或之后。
llen KEY | 获取列表长度
lpop KEY | 移除并返回列表的第一个元素
lpush KEY VALUE1 VALUE2 | 将一个或多个值插入到列表头部
lpushx KEY VALUE | 将一个值插入到已存在的列表头部
lrange KEY START END|返回列表中指定区间内的元素
